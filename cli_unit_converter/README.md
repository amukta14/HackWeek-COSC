# CLI Temp Converter

A simple command-line tool to convert temperatures between Celsius and Fahrenheit.

## Usage

**Celsius to Fahrenheit:**
```bash
python converter.py -c 25
```

**Fahrenheit to Celsius:**
```bash
python converter.py -f 77
```
